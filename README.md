# インスタ リサーチツール

任意のカテゴリ（育児・美容・料理など）のInstagramアカウントをウォッチし、「フォロワー増加」「エンゲージェメント」「バズ投稿」を毎朝自動で集計するツール。**月額0円**で運用。

## 仕組み

- **データ収集**: Bright Data無料枠（月5,000レコード、毎月更新）
- **自動実行**: GitHub Actions（毎朝6:00 JST）
- **保存**: このリポジトリの `data/<カテゴリID>/` にJSONで蓄積
- **閲覧**: `dashboard/` の静的ページ（カテゴリタブで切り替え）

## カテゴリの追加方法

`config/categories.json` の `categories` に追加するだけ：

```json
{
  "id": "biyou",
  "name": "美容",
  "icon": "💄",
  "accounts": ["アカウント名1", "アカウント名2"],
  "hashtags": ["スキンケア", "コスメ好き"]
}
```

次回の収集から自動で対象になり、ダッシュボードにタブが増える。

## 実行方法

```bash
# 全カテゴリ収集
BRIGHTDATA_API_KEY=xxx node scripts/collect.mjs

# 特定カテゴリのみ
BRIGHTDATA_API_KEY=xxx node scripts/collect.mjs ikuji
```

GitHub Actionsで動かす場合はリポジトリのSecretsに `BRIGHTDATA_API_KEY` を登録。

## 無料枠の目安

1アカウントあたり毎日「プロフィール1 + リール10 = 11レコード」消費。
「合計アカウント数 × 11 × 30日 ≤ 5,000」なので、**合計15アカウントまで**無料枠内。
例: 育児5 + 美容5 + 料理5 = 月4,950レコード ✓
アカウントを増やしたい場合はリール収集の頻度を落とす（週1など）か件数を減らす。

## 制限事項

- 他アカウントの**リール再生回数はInstagramの仕様で非公開**（取得不可）。リールランキングはいいね数順で代替
- 取れる指標: フォロワー数、投稿数、エンゲージェメント率、投稿・リールのいいね/コメント数
