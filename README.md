# インスタ キーワードリサーチ

キーワードを入れると、そのジャンルのInstagramアカウントを**フォロワー数順・バズ投稿つき**でリサーチするツール。**月額0円**運用。

## 使い方

1. キーワードを入力して検索（3〜5分）
2. ヒットしたアカウントが**フォロワー順**で表示（バズ投稿順にも切替可）
3. 各アカウントの**バズ投稿TOP3**をリンクからそのままInstagramで見られる
4. 過去の検索結果は履歴からいつでも再表示

## 仕組み

- **検索**: DuckDuckGo等の無料検索で instagram.com 内をキーワード検索
- **データ取得**: Bright Data無料枠（月5,000レコード、毎月更新）→ 検索1回約15レコード＝**月300回以上検索可能**
- **実行**: GitHub Actions（画面から手動起動のみ・自動実行なし）
- **閲覧**: GitHub Pages（`dashboard/`）

## セットアップ（管理者）

1. GitHub Secretsに `BRIGHTDATA_API_KEY` を登録
2. アプリの⚙️からアクセストークン（repo/workflow権限）を設定（端末ごとに1回）

## 制限事項

- 他アカウントの**リール再生回数はInstagramの仕様で非公開**（取得不可）。バズ判定はいいね数
- 検索エンジン経由の発掘のため、ヒットするのは「キーワードに関連して言及・発信しているアカウント」（Instagram内の完全な全文検索ではない）
