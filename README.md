# インスタ リサーチツール

**調べたいときにキーワードで検索**して、そのジャンルのInstagramアカウントと**バズ投稿の上位**をリサーチするツール。**月額0円**で運用。

## 使い方（メイン機能：アカウント発掘）

1. ダッシュボードの「🔍 アカウント発掘」にキーワードを入力（例: スニーカー、車、キャンプ）
2. 3〜5分でそのジャンルのアカウントが**フォロワー順ランキング**で表示
3. 各アカウントの下に**バズ投稿TOP3**（いいね数順・リンク付き）
4. 気になるアカウントは「追加」でウォッチリストへ登録
5. ウォッチリストは「⚙️ → 今すぐ更新」で最新データを手動収集（自動収集は停止中）

## 仕組み

- **検索**: DuckDuckGo等の無料検索で instagram.com 内をキーワード検索（追加のAPIキー不要）
- **データ取得**: Bright Data無料枠（月5,000レコード、毎月更新）
- **実行**: GitHub Actions（画面のボタンから手動起動のみ）
- **保存**: このリポジトリの `data/` にJSON蓄積
- **閲覧**: `dashboard/`（GitHub Pages）

## 無料枠の目安

- 発掘検索1回 ≒ 15レコード → **月300回以上検索可能**
- ウォッチリストの手動更新1回 ≒ アカウント数×11レコード

## セットアップ（管理者）

1. GitHub Secretsに登録: `BRIGHTDATA_API_KEY`
2. アプリの⚙️にアクセストークン（repo/workflow権限）を設定

## 制限事項

- 他アカウントの**リール再生回数はInstagramの仕様で非公開**（取得不可）。バズ判定はいいね数で代替
- 取れる指標: フォロワー数、投稿数、エンゲージェント率、投稿・リールのいいね/コメント数
